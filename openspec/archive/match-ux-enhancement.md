# 相性頁面易讀性優化

## 背景

目前相性診斷結果雖然資料齊全，但使用者反映看不懂。主要問題：
1. 關係方向不直觀 — 只顯示「友衰（衰）」不知道誰對誰
2. 元素關係沒解釋 — 只有元素名稱沒有相生相剋說明
3. API 有回傳 avoid/good_for 但前端沒顯示
4. 缺少白話翻譯 — 需要像「她是友方 = 主動給予的一方」這種解釋

## 變更內容

### 1. 方向解釋區塊（新增）

在「你 ← 關係 → 對方」下方新增方向說明：
- 「你→對方: 衰（被照顧、被影響的一方）」
- 「對方→你: 友（主動給予、照顧的一方）」

每種關係方向加上白話解釋：
- 栄: 帶來好運和正能量的一方
- 親: 親近感強、主動靠近的一方
- 友: 主動給予、照顧的一方
- 成: 被借力、提供價值的一方
- 命: 本命共鳴、鏡像般的關係
- 衰: 被照顧、被影響的一方
- 危: 帶來變化和風險的一方
- 壊: 破壞既有模式的一方
- 安: 被穩定、接受安撫的一方
- 胎: 孕育可能性的一方
- 業: 因果牽連深的一方

### 2. 元素關係說明（新增）

在方向解釋下方顯示元素關係：
- 「木生火 — 對方的能量自然滋養你」
- 「火剋金 — 你的能量會壓制對方」
- 「同元素（火）— 能量共鳴，互相增強」

### 3. 顯示 avoid 列表

在「小技巧」下方新增「避免事項」列表，顯示 relation.avoid 資料。

### 4. 顯示 good_for 列表

在角色別指南上方新增「適合場景」標籤，顯示 relation.good_for 資料。

### 5. 我的配對列表也同步更新

partner-detail 展開後同樣顯示方向解釋、元素關係、avoid、good_for。

## 影響範圍

| `frontend/src/components/MatchTab.vue` | 新增方向說明、元素關係、avoid/good_for 顯示 |
