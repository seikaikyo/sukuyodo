# 知識補充與特殊日/九曜說明強化

## 問題

1. **甘露日/金剛峯日/羅刹日看不到**：只在當天是特殊日時才顯示 banner，平時完全無痕跡，使用者不知道有這個功能
2. **知識 tab 沒補充**：新增了甘露日和九曜流年法，但知識 tab 完全沒有這兩個主題的說明
3. **本地佛沒有解釋**：年運勢顯示「本地佛：千手觀音」但沒解釋這是什麼

## 變更內容

### 1. 每日運勢增加特殊日說明區

- 在日運勢面板底部（或適當位置）加一個靜態的小說明區，解釋甘露日/金剛峯日/羅刹日
- 不是特殊日時也能看到這三種日的定義
- 類似「今日非特殊日」或「本週特殊日：2/23 甘露日」的預告

### 2. 知識 tab 新增「特殊日」子區塊

- 在知識 tab 加入甘露日/金剛峯日/羅刹日的完整說明
- 包含：定義、吉凶等級、七曜 x 宿的對照表、適合/不適合做的事

### 3. 知識 tab 新增「九曜」子區塊

- 九曜流年法的說明：什麼是九曜、循環規則、數え年計算
- 九顆星各自的意義、吉凶、元素
- 本地佛（本地仏）的概念說明：每顆九曜星對應的守護佛

### 4. 年運勢「本地佛」加上 tooltip 或說明

- 「本地佛」旁加一句簡短解釋：「九曜星對應的守護佛尊」
- 或改為更易懂的標籤如「守護佛」

## 影響範圍

| 檔案 | 修改內容 |
|------|---------|
| `backend/data/sukuyodo_mansions.json` | metadata 新增特殊日和九曜知識段落 |
| `frontend/src/components/KnowledgeTab.vue` | 新增特殊日和九曜子 tab |
| `frontend/src/components/FortuneTab.vue` | 日運勢加特殊日說明、年運勢本地佛改善 |
| `frontend/src/composables/useSukuyodo.ts` | metadata 型別更新（如需） |

## 不修改

- 所有運算邏輯
- 後端 API 端點
- 既有的運勢文案

## 測試計畫

1. 知識 tab 新增子區塊正常顯示
2. 日運勢特殊日說明可見
3. 年運勢本地佛標籤清楚易懂
4. 本機 pm2 restart 驗證
