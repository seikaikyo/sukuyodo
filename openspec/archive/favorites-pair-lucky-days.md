# 收藏對象 + 雙人吉日功能

## 變更內容

### 1. 擴充個人吉日類別

新增更多個人吉日項目：
- 美容：理髮、染髮、燙髮、美甲、護膚
- 購物：大額消費、買衣服、買首飾

### 2. 收藏對象管理

新增/編輯/刪除收藏對象，包含：
- 暱稱
- 生日
- 關係類型（交往對象、配偶、父母、家人、朋友/同事）

### 3. 雙人吉日

根據關係類型顯示對應的共同吉日：
- 交往對象：約會、告白、見家長、訂婚、結婚登記、婚禮
- 配偶：約會、旅遊、重要商量
- 父母：探親、送禮、重要商談
- 家人：聚會、旅遊
- 朋友/同事：聚會、合作

## 影響範圍

- `frontend/src/stores/profile.ts` - 增加關係類型
- `frontend/src/components/LuckyDaysTab.vue` - 重新設計 UI
- `frontend/src/composables/useSukuyodo.ts` - 新增雙人吉日邏輯
- `backend/routers/sukuyodo.py` - 新增雙人吉日 API
- `backend/services/sukuyodo.py` - 雙人吉日計算邏輯
