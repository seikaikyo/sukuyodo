# 日本選日曆注功能

## 變更內容

新增日本傳統「選日」系統，與現有宿曜吉日並列顯示但明確區隔。

### 選日項目

| 項目 | 日文 | 說明 | 計算方式 |
|------|------|------|---------|
| 一粒萬倍日 | 一粒万倍日 | 播種一粒收萬倍，適合開業、存錢 | 干支循環 |
| 天赦日 | 天赦日 | 一年約 5-6 天，萬事皆吉 | 季節 + 干支 |
| 寅の日 | 寅の日 | 金運吉日，12 天一次 | 十二支 |
| 巳の日 | 巳の日 | 財運（弁財天），12 天一次 | 十二支 |
| 己巳の日 | 己巳の日 | 最強金運日，60 天一次 | 干支 |
| 不成就日 | 不成就日 | 凶日，避免重要事項 | 固定週期 |

### 六曜

| 項目 | 說明 |
|------|------|
| 大安 | 萬事大吉 |
| 友引 | 午時凶，其餘吉 |
| 先勝 | 上午吉 |
| 先負 | 下午吉 |
| 赤口 | 正午吉，其餘凶 |
| 仏滅 | 凶日 |

## 影響範圍

- `backend/services/japanese_calendar.py` - 新增：選日計算服務
- `backend/routers/sukuyodo.py` - 新增 API 端點
- `frontend/src/components/LuckyDaysTab.vue` - UI 區隔顯示
- `frontend/src/composables/useSukuyodo.ts` - 新增型別和 API 呼叫

## UI/UX 規格

### 介面區隔

兩個區塊使用不同的左邊框顏色區分：
- 宿曜吉日：紫色 (#8b5cf6)
- 選日曆注：朱紅色 (#dc2626)

### 設計要點

1. **視覺區隔**：用不同底色和邊框區分兩個系統
2. **來源標示**：各區塊標題明確
3. **說明文字**：宿曜基於本命宿，選日與生日無關
4. **超吉日標記**：多重吉日疊加時有金色邊框和星號標示

## API 端點

| 端點 | 方法 | 說明 |
|------|------|------|
| `/calendar/lucky-days/{year}/{month}` | GET | 取得整月選日資料 |
| `/calendar/day/{date}` | GET | 取得單日選日資訊 |
| `/calendar/upcoming` | GET | 取得未來 N 天吉日 |
