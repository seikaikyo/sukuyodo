# 修正每日運勢描述文字

## 問題描述

每日運勢顯示的關係描述使用了雙人配對的文字，例如：
- 「兩人之間存在明顯的權力不對等」
- 「最適合結婚的對象」

這些文字在每日運勢的語境下完全不合理，讓使用者困惑。

## 解決方案

為每日運勢新增專用的描述文字，區分「本命宿 vs 當日宿」的情境。

## 影響範圍

- `backend/services/sukuyodo.py` - 修改 `calculate_daily_fortune` 方法

## 新增描述對照

| 關係類型 | 原雙人配對描述 | 新每日運勢描述 |
|---------|--------------|---------------|
| 榮親 | 最適合結婚的對象... | 大吉之日，貴人運旺，適合積極行動、開展新事物 |
| 業胎 | 前世因緣深厚... | 吉日，直覺敏銳，適合創意發想與深度思考 |

| 危成 | 互補的關係... | 小心之日，行事需謹慎，注意細節 |
| 安壞 | 權力不對等... | 挑戰之日，重大決定宜延後，保持低調 |
