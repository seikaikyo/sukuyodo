# 密教修法擇日系統

## 背景

宿曜經（T21n1299）的「本業」是密教修法擇日，不是一般人的占卜。不空三藏編纂此經的核心目的是為灌頂、護摩、受戒、造像等密教修法提供權威的擇日手冊（Jeffrey Kotyk, Leiden 2017）。

目前系統的問題：**後端已有極豐富的修行資料（~500 行），前端顯示率 0%**。

| 後端資料 | 程式碼行 | 前端使用率 |
|---------|---------|-----------|
| 9 年修法框架（SHINGON_KUYOU_DATA） | 2528-3048 | 0% |
| 護摩種類與時機（息災/增益/降伏） | 散布多處 | 0% |
| 修行月度里程碑（12 法會日） | 2569-2580 | 0% |
| 特殊日修法用途描述 | 1108-1126 | 僅純文字 |
| 剃髮修行禮儀 | 3736-4100 | 藏在吉日查詢 |

## 變更內容

### Phase 1: 年運勢修行區塊強化（最小修改，最大效益）

**策略：不新增 Tab，而是在既有的年運勢頁面中突出已有的修行資料。**

年運勢 API 回傳的 `kuyou_star` 已包含完整的 `practice_*` 欄位，只是前端沒顯示。

#### 1-1 前端：FortuneTab.vue 年運勢區塊

在年運勢的九曜流年資訊下方，新增「修行指南」區塊：

- **年度修行主題**：practice_name + practice_focus
- **推薦修法**：recommended_practices 列表
- **護摩指引**：homa_type + homa_description
- **月度里程碑**：monthly_milestones 12 項法會日期

#### 1-2 前端：FortuneTab.vue 每日運勢區塊

在特殊日（甘露/金剛峯/羅刹）的顯示中，強化修法指引：

- 甘露日：「宜受灌頂法、受戒、習學經法」（品第八原文）
- 金剛峯日：「宜作降伏法、誦日天子呪、作護摩」（品第八原文）
- 羅刹日：「不宜舉百事」+ 替代建議（靜坐、誦經迴向）
- 凌犯 + 甘露日逆轉時：警示「甘露日被凌犯，修法宜慎重」

#### 1-3 前端：每日運勢的三九法修行提示

根據當日的三九法關係，顯示修行相關提示：

- 榮宿日：「出家人剃髮、承事師主、啟請法要並吉」（卷下原文）
- 安宿日：「作壇場並吉」
- 成宿日：「修道學問、作諸成就法並吉」
- 命/胎宿日：「不宜舉動百事」
- 危壞日：「不宜剃頭、沐浴」

### Phase 2: 後端 API 修行資料補充（視需要）

目前年運勢 API 已回傳 `kuyou_star` 中的修行欄位，每日運勢也有 `special_day` 和 `relation_type`。

如果 Phase 1 發現資料不足，再考慮：
- 每日運勢新增 `practice_hint` 欄位（三九法修行提示）
- 吉日查詢新增 `shuho`（修法）分類

## 設計原則

1. **經文為本**：所有修行指引必須引用 T21n1299 原文出處
2. **不造假**：不編造修法建議，僅呈現經文已有的擇日規則
3. **漸進式**：先用既有 API 資料，前端顯示為主，避免大改後端
4. **真言宗視角**：以密教修行者為主要受眾，不是一般占卜用戶

## UI/UX 規格

### 年運勢修行區塊
- 位置：九曜流年星資訊下方
- 風格：與現有年運勢風格一致
- 色彩：使用既有的 element color 系統
- 間距：4px 倍數（8/12/16/24px）
- 響應式：mobile 一欄，desktop 雙欄

### 每日特殊日修法指引
- 位置：特殊日標記旁
- 風格：引用框（blockquote 風格），標注出處品第
- 不使用 emoji
