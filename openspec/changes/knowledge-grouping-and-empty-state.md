# 知識 tab 分組、空狀態改善、關係手風琴

## 變更內容

### 1. 知識 tab 10 個子分頁分組

10 個 pill 按鈕排在一行，認知負擔過高。改為分類標籤顯示：

- **基礎**：本命宿 / 二十七宿 / 六種關係 / 五行七曜
- **宿曜曆法**：特殊日 / 九曜 / 凌犯逆轉 / 三九秘曆 / 傍通曆
- **背景**：歷史

每組前方顯示分類標籤，pills 按組排列。

### 2. 首頁空狀態改善

將「了解你的命宿特質與每日運勢」改為更具說明性的文字，
讓首次使用者理解宿曜道是什麼系統。

### 3. 六種關係改為手風琴

目前知識 tab 的六種關係全部展開，頁面很長。
改為手風琴（accordion）：預設全部收合，點擊標題展開內容。

## 影響範圍

| 檔案 | 修改 |

| `frontend/src/components/KnowledgeTab.vue` | tab 分組 + 關係手風琴 |
