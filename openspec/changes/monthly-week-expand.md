# 本月運勢週次展開功能

## 變更內容

在本月運勢頁面，點擊週次可展開顯示該週的詳細運勢資料。

## 影響範圍

- `frontend/src/components/FortuneTab.vue` - 新增週次點擊展開 UI
- `frontend/src/composables/useSukuyodo.ts` - 新增展開狀態與週運勢載入邏輯

## UI/UX 規格

### 互動設計

```
┌─────────────────────────────────────┐
│ 每週概覽                             │
├─────────────────────────────────────┤
│ [第 1 週] ████████░░ 80              │ ← 可點擊
│ [第 2 週] ██████░░░░ 65              │ ← 可點擊
│ ▼ 第 3 週 ████████████ 92 (本週)     │ ← 展開中
│   ┌─────────────────────────────┐   │
│   │ 整體 92  事業 88  感情 75    │   │
│   │ 每日: 一85 二78 三92 ...    │   │
│   │ 建議: 本週適合...            │   │
│   └─────────────────────────────┘   │
│ [第 4 週] ███████░░░ 72              │
└─────────────────────────────────────┘
```

### 設計規格

| 項目 | 規格 |
|------|------|
| 點擊區域 | 整個週次列可點擊 |
| 展開動畫 | height transition 200ms ease |
| 展開指示 | 左側 ▶/▼ 圖示切換 |
| 當前週標記 | 加上 (本週) 標籤 |

## 測試計畫

1. 點擊週次可展開/收合
2. 同時只能展開一個週次
3. 展開後載入該週詳細運勢
4. 鍵盤可操作（Enter/Space 展開）
