# 知識輔助迷你卡片

## 變更內容

兩個部分：

### A. 輪盤下方即時卡片
根據輪盤當前顯示的層級，自動切換對應的迷你卡片說明：
- 預設（元素色）→ 五行元素卡片
- 七曜環可見 → 七曜卡片
- 三期弧形可見 → 三九秘曆卡片
- 關係全覽 ON → 六種關係卡片
- 凌犯狀態 → 凌犯逆轉卡片

卡片格式：標題 + 2-3 行說明 + 視覺提示（色塊）+ 「了解更多」連結跳到知識頁籤

### B. 知識頁籤頂部導讀
每個知識頁籤（九曜、凌犯逆轉、三九秘曆、七科分宿等）頂部加一段 1-2 行摘要導讀。

## 影響範圍
- `frontend/src/components/KnowledgeTab.vue`（主要修改）

## 測試計畫
1. `npm run build` 通過
2. 輪盤頁：預設顯示五行卡片，開啟全覽顯示關係卡片
3. 各知識頁籤頂部有導讀文字
4. 「了解更多」連結正確跳轉到對應頁籤
5. mobile 響應式正常
